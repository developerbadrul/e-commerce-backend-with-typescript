API Documentation

API Endpoints

Endpoint Method Description
/api/products POST Create a New Product
/api/products GET Retrieve a List of All Products
/api/products/:productId GET Retrieve a Specific Product by ID
/api/products/:productId PUT Update Product Information
/api/products/:productId DELETE Delete a Product
/api/products?searchTerm=iphone GET Search a Product
/api/orders POST Create a New Order
/api/orders GET Retrieve All Orders
/api/orders?email=level2@programming-hero.com GET Retrieve Orders by User Email

Product Management

1. Create a New Product

Endpoint: https://assignment-2-p-hero.vercel.app/api/products

Method: POST

Sample Request Body:

{
    "name": "iPhone 13",
    "description": "A sleek and powerful smartphone with cutting-edge features.",
    "price": 999,
    "category": "Electronics",
    "tags": ["smartphone", "Apple", "iOS"],
    "variants": [
        {
            "type": "Color",
            "value": "Midnight Blue"
        },
        {
            "type": "Storage Capacity",
            "value": "256GB"
        }

    "inventory": {
        "quantity": 50,
        "inStock": true
    }
}

Order Management

1. Create a New Order

Endpoint: https://assignment-2-p-hero.vercel.app/api/orders

Method: POST

Request Body:

{
    "email": "level2@programming-hero.com",
    "productId": "5fd67e890b60c903cd8544a3",
    "price": 999,
    "quantity": 1
}

Sample Response:

{
    "success": true,
    "message": "Order created successfully!",
    "data": {
        "email": "level2@programming-hero.com",
        "productId": "5fd67e890b60c903cd8544a3",
        "price": 999,
        "quantity": 1
    }
}
